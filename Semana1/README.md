Basicamente, tenemos que un **HOLA MUNDO** en Python es de la siguiente forma:

```python
def main():
    print("Hola mundo")

main()
```

En cambio, en Java es:

```java
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("Hola mundo");
    }
}

```
